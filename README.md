工业大数据是未来全球竞争的关键。无论是德国工业4.0、美国工业互联网还是《中国制造2025》, 制造业创新战略的基础都是数据的采集和分析, 而数据采集则是其先决条件。物联网数据采集通过物联网硬件对开关、模拟量数据进行采集, 实现对温度、震动及环境等多种数据的采集; 通过云平台进行远程数据监测从而实现远程控制, 或通过移动终端进行远程监控。

本课程讲授数据采集与物联网的理论及应用架构，穿插应用案例进行分析，介绍如何搭建开源的开发环境及软件、固件开发实战操作。

数据采集是测量电、声、温、压等物理世界条件和现象的过程。使用传感器采集环境中的模拟信号，再使用模数转换将其转换为数字信号。然后生成的数字数据通过物联网（IoT）传输到数据中心，进而作为大数据通过云计算系统对数据集进行操作、分析、存储和表示。

数据采集需要四个组件：

1） 能够捕捉环境模拟信号的传感器，如温度、压力、光或声音

2） 使捕获的信号标准化的信号调节电路；降噪器和放大器是很好的例子

3） 模数转换器，将模拟信号转换成数字数据

4） 物联网设备将数字数据传输到远程数据中心
